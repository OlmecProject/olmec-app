# olive-front

## Project setup
```
yarn install
```

### Compile for Electron
```
yarn electron:serve
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```
